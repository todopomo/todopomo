What is TodoPomo?

TodoPomo is a timer plus todo list tool that uses pomodoro technique. It uses timer to divide tasks into intervals. The task timer is usually 25 minutes (customizable) and the break timer is usually 5 mins long(customizable). After completion of every task interval, you get a break.

How to use this tool?

TodoPomo is a simple to use tool. You can start the task timer with the time of your choice. Add todo lists and complete the tasks. Task timer as well as break timer are customizable.
